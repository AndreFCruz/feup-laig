# Project 3 - Fabrik
### Usage
After cloning the repository, to start the javascript client simply run the ```start_client.sh``` script
```shell
sh start_client.sh
```

...and to start the prolog server, change directory - ```cd TP3``` - to TP3 and run the ```start_server.sh``` script
```shell
sh start_server.sh
```
